# Dashboard Data Jember

Dashboard visualisasi data per kecamatan di Kabupaten Jember.

## 📊 Features
- Visualisasi data menggunakan Chart.js
- Export grafik ke PDF
- Tampilan responsive
- Ringkasan data per kecamatan

## 🛠️ Tech Stack
- HTML
- CSS / Tailwind
- JavaScript
- Chart.js
- jsPDF

## 📌 Tujuan Project
Project ini dibuat untuk menampilkan data statistik Kabupaten Jember dalam bentuk visual yang informatif dan mudah dipahami.
